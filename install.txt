EBSD ANALYSIS MODULE

Подготовка:
1. Модуль расчитан на работу с MATLAB 7.12.0 (R2011a)
2. Для работы модуля необходимо установить и модернизировать пакет MTEX 3.3.2
2.1 Скачайте модуль MTEX
2.2 Скопируйте в отдельную папку, например:
	C:\MATLAB\MTEX\
2.3 Запуските MATLAB с правами администратора (актуально для Windows Vista, Windows 7)
2.4 Перейдите в папке C:\MATLAB\MTEX\ и запустите 'startup_mtex' (install MTEX permanent - yes)
2.5 Переименуйте файл 'startup.m' в 'startup-old.m'
3. Если была установлена предыдущая версия EBSDAM, ее на до удалить (попробуйте запустить 'uninstall_ebsd' или 'startup_ebsdam('uninstall')').


Установка:
1. Скопируйте все файлы в отдельную папку, например:
	C:\MATLAB\EBSDAM\
2. Запуските MATLAB с правами администратора (актуально для Windows Vista, Windows 7)
3. Перейдите в папке C:\MATLAB\EBSDAM\ и запустите 'startup'
3. Настройте рабочии папки, для этого запустите 'setupWorkDirs'
3.1 DATA_DIR   - папка с EBSD данными
3.2 OUTPUT_DIR - папка для результатов (в ней будет создана папка img)
3.3 CACHE_DIR  - папка для сохранения промежуточных данных
3.4 DOC_DIR    - папка для отчетов и документов (на данный момент не используется)
4. Проверьте установку 'check_ebsdam'


Советы:
1. Для удобства последуюшего обновления все произвождые файлы лучше хранить в отдельной папке.
   Данную папку нужно будет выбрать рабочей в MATLAB.
