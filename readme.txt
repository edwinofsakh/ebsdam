Для начала работы запуските Matlab с правами администратора и выполните скрипт 'startup_ebsdam'.
Для загрузки новых данных смотрите "\load\readme.m"

Полезные функции:
viewMaps - Карты ориентировки
viewPhases - Информация о фазах
viewProp - Дополниетельные свойства, например: качество изображения, высота.
viewGrains - Карты зерен
viewSizes - Информация о размере зерен
viewAll - Выполнение всех предыдущих функций

Example:
viewAll( 's04', 0, 1, 5, 8 )
Вся информация об образце 's04'.
Загруженых ESBD данных '0'.
Сохранять изображения '1'.
Порог определяние границы зерна '5' градусов.
Зерна меньше '8' точек удаляются.
